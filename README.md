# Stopwatch

シンプルなブラウザ向けストップウォッチ。HTML / CSS / JavaScript のみで構成された静的サイトです。

## 機能

- スタート / ストップ
- リセット
- ラップ計測（直近ラップとの差分も表示）
- センチ秒（1/100 秒）まで表示
- ダークテーマ UI

## ローカルでの起動

依存関係はありません。`index.html` をブラウザで直接開いてください。

```bash
# Windows
start index.html
```

## デプロイ

[Vercel](https://vercel.com/) で公開しています。`index.html` がリポジトリのルートにあるため、Framework Preset を `Other` にし、Build / Output 設定は空のままで Deploy できます。
